Работаем с переменными и константами

На этом уроке мы узнаем, что такое переменные и константы, а также напишем программу, которая их использует.

Что такое переменные?

Переменная — это как коробка с именем, в которой можно хранить значение: текст, число или многое другое. Вы можете изменять содержимое коробки в любой момент. Но в ней может храниться только одно значение. Это значение можно читать или изменять, поэтому переменная и называется переменной. При сохранении нового значения старое теряется.

name = "Алиса"

В этом примере мы создали переменную name и положили в нее текст "Алиса".

Имя переменной еще называют ее идентификатором. Под коробкой имеется в виду область машинной памяти, где хранится значение.

Когда в программе мы указываем имя переменной, компьютер "знает", где найти эту область памяти по имени.

Что такое константы?

Константа похожа на переменную, но её значение нельзя изменить. Это как коробка, которую запечатывают навсегда. В Python принято использовать заглавные буквы для имен констант:

PI = 3.14

Здесь PI - именованная константа (имеющая имя). Если в программе нам надо вычислить длину окружности по ее радиусу r, мы можем сделать это, как используя само число Пи, так и имя константы, которая хранит значение числа Пи:

circle_length = 2 * r * PI  # Используем имя константы
circle_length = 2 * r * 3.14 # Используем значение константы

Есть также неименованные константы, например числа или строки, которые сразу используются в коде: print(42) или print("Привет").

В этом коде 42 - неименованная числовая константа, а "Привет" - неименованная строковая.

Функция print()

Подробнее о функциях мы поговорим позже. Для начала познакомимся с одной из них, которая выводит значение на экран. Чтобы сделать это, напишите слово "print", за которым в круглых скобках укажите значение или выражение:

print("Привет, Том!")

Строка "Привет, Том!" - это константа. Когда она употребляется в круглых скобках после имени функции, говорят, что она передается в функцию print для вывода на экран. При этом значение, передаваемое в функцию, называется параметром функции. В функцию print можно передать не один, а несколько параметров, которые разделяются запятыми:

name = "Том"
print("Привет, ", name)
print("Привет, ", name, ". Как дела?")

Пример программы

Вот пример программы, которая демонстрирует использование переменных и констант:

# Объявляем переменную
name = "Алиса"

# Объявляем константу
PI = 3.14

# Выводим значения на экран
print("Привет,", name)
print("Значение PI:", PI)

# Изменяем значение переменной
name = "Боб"
print("Теперь имя:", name)
