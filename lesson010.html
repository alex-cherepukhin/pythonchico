Отступы. Условная конструкция if.

Изучим что такое отступы, и научим программу гибко реагировать в зависимости от входных данных.

Что такое отступы?

Программа на языке Python состоит из набора инструкций. Каждая инструкция помещается на новую строку. Например:

print(2 + 3) 
print("Hello")

Отступ - это пробелы в начале строки кода. В Пайтоне каждая новая инструкция (команда) должна начинаться с начала строки, без отступа. Например, в следующем случае мы получим ошибку, хотя код будет практически аналогичен приведенному выше:

print(2 + 3) 
    print("Hello") # Ошибка!

Однако стоит учитывать, что некоторые конструкции языка могут состоять из нескольких строк. Например, условная конструкция if:

if 1 < 2:
    print("Hello!")

В данном случае если 1 меньше 2, то выводится строка "Hello". И здесь уже должен быть отступ, так как инструкция print("Hello") используется не сама по себе, а как часть условной конструкции if. В качестве отступа надо использовать четыре пробела.

Конструкция if

Еще её называют условным оператором. Самая простая конструкция if состоит из:

  • Ключевого слова if
  • Условия, за которым следует двоеточие
  • Последующего блока инструкций, каждая из которой должна начинаться с новой строки и должна иметь отступ от начала выражения if.

language = "english"
if language == "english":
    print("Hello!")
    print("How do you do?")

Значение переменной language равно "english", поэтому условие в выражении if истинно, а значит выполнится инструкция print(), которая выведет на экран приветствие по-английски.

Hello!
How do you do?

Давайте изменим код программы

language = "english"
if language == "spanish":
    print("¡Hola!")
print("¿Como estas?")

Условие в выражении if перестало быть истинным, поэтому первое сообщение не будет выведено на экран. Но перед второй инструкцией print() нет оступа от начала выражения if, поэтому она не относится к условному оператору и будет выведена на экран вне зависимости от условия if:

¿Como estas?

Блок else

Если нам надо выполнить альтернативное действие на тот случай, если выражение в условии if возвратит False (ложь), то мы можем использовать блок else:

language = "english"
if language == "spanish":
    print("¡Hola!")
else:
    print("Привет!")

Если выражение language == "spanish" возвращает True, то выполняется блок if, иначе выполняется блок else. И поскольку в данном случае условие language == "spanish" возвращает False, то будут выполняться инструкция из блока else.

Привет!

Блок elif

Если необходимо ввести несколько альтернативных условий, то можно использовать дополнительные блоки elif, после которого идет блок инструкций.

language = "english"
if language == "spanish":
    print("¡Hola!")
elif language == 'english':
    print("Hello!")
else:
    print("Привет!")

Программа выдаст на экран:

Hello!
