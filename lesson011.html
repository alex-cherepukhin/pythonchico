Работа со строками

Познакомимся с функциями и методами строк.

Базовые действия со строками

Конкатенация или сложение строк - это операция соединения строк в одну. Используется операция "+"

k = "кот"
print(k + "лета") # результат: "котлета"

Дублирование (размножение) строки, операция "*"

print("Бла" * 3) # Результат: "БлаБлаБла"

Получение длины строки, функция len()

print(len("питон")) # Результат: 5

Преобразование строки к верхнему регистру, метод upper()

msg = "Тихо, дети!"
print(msg.upper()) # Выдаст на консоль: "ТИХО, ДЕТИ!"

Преобразование строки к нижнему регистру, метод lower()

print("АаАаОоОо".lower()) # Результат: "ааааоооо"

Операции, функции и методы строк

Давайте научимся понимать, чем отличаются эти понятия. Операции отделяются от самой строки при записи и обозначаются значками. Это операции + и *.

"доедай" + "додыр" #Операция + конкатенирует строки

Функция - это самостоятельный блок кода, который выполняет определенную задачу и имеет имя. Функции вызываются напрямую по своему имени, и им могут передавать аргументы. Например, передаем строку в качестве аргумента фукнции len(). Функция len() при этом не принадлежит самой строке, а существует независимо.

l = len("питон") # Функция len() вычисляет длину строки
s = str(l) # Функция str() преобразует число в строку
print("Длина строки = "+s) # Функция print выводит строку на экран

Методы - это фунции, связанные с определенным объектом (в данном случае, со строкой). Дело в том, что в Пайтоне каждая строка это не просто строка, а объект, у которого есть методы (функции, принадлежащие ему). Чтобы вызвать метод, надо после имени объекта (после переменной, содержащей строку, или после самой строки) поставить точку, и сразу за ней написать имя метода. Например:

s = "abc".upper() # В переменную s запишется "ABC"
print(s.lower()) # Будет напечатано: abc

Методы нельзя употреблять сами по себе, без префикса объекта, которому метод принадлежит:

s = upper("abc") # Ошибка!

Какие ещё есть методы у строк

Метод capitalize() переводит первый символ строки в верхний регистр, а все остальные в нижний

print("конечно ВАСЯ".capitalize()) # Результат: "Конечно вася"

Метод title первую букву каждого слова переводит в верхний регистр, а все остальные в нижний

print("каждый охотник".title()) # Результат: "Каждый Охотник"

Метод isupper() проверяет, что в строке есть буквы в ВЕРХНЕМ регистре, но нет букв в нижнем регистре, и возвращает True или False

print("СТОП!".isupper()) # Результат: True (с русскими буквами - работает)
print("S T.O?P:)".isupper()) # Результат: True (с латинскими буквами - тоже работает)
print('123'.isupper()) # Результат: False (нет букв)
print("QWERTy".isupper()) # Результат: False (есть буква в нижнем регистре)

Метод islower() проверяет, что в строке есть буквы в нижнем регистре, но нет букв в ВЕРХНЕМ регистре. Она также возвращает True или False

print("bubble gum".islower()) # Результат: True
print("""Привет Китти""".islower()) # Результат: False

Метод isdigit() проверяет, состоит ли строка из одних цифр

print("-1".isdigit())    # False
print("1.23".isdigit())  # False
print("123".isdigit())   # True
print(" 123 ".isdigit()) # False
print("".isdigit())      # False

Метод isalpha() проверяет, состоит ли строка из одних букв

print("ПреттиGood".isalpha()) # True
print("раз два".isalpha())   # False

Метод isalnum() проверяет, состоит ли строка из цифр или букв

print("abc123АБВ".isalpha()) # True

Есть еще много методов у строк, но на этот раз не буду вас ими мучить )
