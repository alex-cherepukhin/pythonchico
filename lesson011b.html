Разбор задачи урока 11

Задача оказалась нелегкой и решение неочевидным. Разберем его.

Условие задачи №2

Пользователь вводит строку. Программа должна проверить, что в ней есть буквы как в вернем, так и в нижнем регистре. Надо вывести результат проверки (True / False).

Пояснения к условию.

Буквы в верхнем регистре - это заглавные буквы (АБВ), буквы в нижнем регистре - это строчные (абв).

Разделим задачу на отдельные шаги, каждый из которых понять и выполнить легче, чем целую задачу.

  1. Ввести строку с клавиатуры и присвоить ее строковой переменной.
  2. Определить, есть ли в строке заглавные буквы.
  3. Определить, есть ли в строке строчные буквы.
  4. Вычислить единый результат проверки (совместить две предыдущих проверки).
  5. Вывести результат на экран.

1. Ввести строку.

Это мы уже делали не раз:

s = input("Введите строку:")

2. Определить, есть ли в строке заглавные буквы.

Сначала рассмотрим несколько примеров строк, чтобы лучше понять задачу

  • "Winnie the Pooh" - есть две заглавные буквы
  • "piglet" - нет заглавных букв
  • "OWL" - все буквы заглавные

Как проверить, что заглавные буквы есть? Если попробовать использовать метод isupper(), это не выход, т.к. на результат влияет наличие строчных букв:

print(isupper("Winnie the Pooh")) # False

Но можно использовать один из методов, которые воздействуют на заглавные буквы, и проверить результат применения метода. Если результат не изменился, значит заглавных букв в исходной строке не было. Используем метод lower()

print(lower("Winnie the Pooh")) # winnie the pooh

Осталось только сравнить с исходной строкой:

s = "Winnie the Pooh"
print(lower(s) != s) # True

После применения метода lower() значение строки изменилось (было "Winnie the Pooh", стало "winnie the pooh"), поэтому проверка на неравенство выдаст True

3. Определить, есть ли в строке строчные буквы.

Это зеркальная задача, она решается точно также, только с применением метода lower(), обратного к методу upper().

s = "Winnie the Pooh"
print(lower(s) != s) # True

4. Совместить две предыдущих проверки.

Если надо проверить, что условие1 истинно И условие2 истинно, используется логический оператор and:

print(True and True) # True

Оператор and совмещает два логических выражения. Подставим сюда выражения, проверяющие исходную строку на наличие заглавных и строчных букв.

s = "Winnie the Pooh"
print(upper(s) != s and lower(s) != s) # True

5. Осталось добавить ввод строки пользователем.

Окончательный результат:

s = input("Введите строку:")
print(upper(s) != s and lower(s) != s) # True

Вторая задача решена!
