F-строки и R-строки

Познакомимся с внедрением переменных в строку, неформатированными строками и понятием кода символа.

F-строки

Их еще называют форматированные строки. Начнем с примера, в котором на экран выводятся значения двух переменных

latitude = 51.9
longitude = 39.3
print(f"Клад закопан тут: широта {latitude}, долгота {longitude}")
# результат: "Клад закопан тут: широта 51.9, долгота 39.3"

Python позволяет встравивать в строку значения других переменных. Для этого внутри строки переменные размещаются в фигурных скобках {}, а перед всей строкой ставится символ "f". F-строку можно присвоить переменной, а не только передать в функцию print(), например:

userName = "Tom"
userAge = 37
user = f"name: {userName}  age: {userAge}"
print(user)   # name: Tom  age: 37

R-строки

Мы уже проходили, как вставить в строку переход на новую строку, используя управляющую последовательность \n. Управляющая последовательность состоит из знака обратного слэша "\", за которым следует символ. При этом говорят, что обратный слэш экранирует этот символ.

text = "Message:\n\"Hello World\""
print(text)

На экран будет выведено:

Message:
"Hello World"

При этом последовательность \n превратилась в перевод строки, а последовательность \" - в двойную кавычку. Но если перед строкой, содержащей управляющие последовательности, поставить символ "r", то превращение не происходит, а последовательности остаются в строке в неизменном виде:

text = r"Message:\n\"Hello World\""
print(text)

Такую строку с символом "r" перед ней ещё называют неформатированной строкой, при этом говорят, что R-строка подавляет экранирование символов, содержащихся в ней. На экран будет выведено:

Message:\n\"Hello World\"

Для примера, переменная path содержит некоторый путь к файлу, и мы хотим вывести его на экран:

path = "C:\python\name.txt"
print(path)

Вот что мы увидим:

C:\python
ame.txt

Внутри строки были символы "\n", которые заменились на перевод строки. Чтобы избежать подобной ситуации, перед строкой ставится символ "r":

print(r"C:\python\name.txt")

Коды символов

У каждого символа (буквы, цифры, пробела, других знаков), который мы используем в тексте программы и в том числе в строках, есть свой номер. Это порядковый номер символа в таблице символов (которая еще называется таблицей ASCII). Номер символа обычно называют кодом символа. Вот коды некоторых символов

Символ Код (десятичный) Описание
A 65 Заглавная буква A
a 97 Строчная буква a
0 48 Цифра 0
9 57 Цифра 9
! 33 Восклицательный знак
@ 64 Символ "собака" (@)
# 35 Решётка (#)
$ 36 Знак доллара ($)
' ' 32 Пробел
\n 10 Перевод строки (LF)

Функция ord() выдает код символа, который передается ей в качестве параметра:

print(ord('a'))

Будет напечатано:

97

Функция chr() действует наоборот, она принимает код символа, и возвращает символ, который имеет такой код (т.е. выдает символ по его коду)

print(chr(97))

Будет напечатано:

'a'

Функции ord() и chr() являются обратными друг ко другу, т.е. если применить одну из них к другой, то на выходе получим исходный символ или исходный код символа:

print(ord(chr(97))) # результат: 97
print(chr(ord('a'))) # результат: 'a'

Ещё важно понять, что эти функции преобразуют значение одного типа в значение другого типа. ord() преобразует символ в число, а chr() - число в символ.

Задания

  2. Пользователь вводит код символа. Надо выдать на экран символ с таким кодом, подавив его экранирование при выводе.
