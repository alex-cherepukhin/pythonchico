Арифметические операции с присвоением

Изучим, как в Пайтоне можно сокращенно записывать некоторые действия. Ряд специальных операций позволяют использовать присвоить результат операции первому операнду:

+= (Присвоение результата сложения)

Предположим, что переменная counter используется как счетчик, т.е. в ней мы храним количество и по ходу программы увеличиваем ее на 1:.

counter = counter + 1

Сократим запись, используя операцию сложения с присвоением:

counter += 1

Операция += суммирует значение переменной (слева) со значением выражения (справа) и присваивает результат переменной (слева). Пример программы:

counter = 0 # Объявляем переменную counter
counter += 1   # Увеличиваем её на 1
print(counter) # Будет напечатано: 1

Слева от операции с присвоением должна быть переменная. Так неправильно:

print(0 += 1)  # Ошибка!

Переменная, используемая в операции с присвоением, должна быть ранее объявлена (т.е. должна уже иметь значение). Нельзя в одной команде объявить переменную и использовать ее значение:

counter += 1   # Ошибка! counter не объявлена
print(counter)

-= (Присвоение результата вычитания)

Операция -= вычитает из значения переменной (слева) значение выражения (справа) и присваивает результат переменной (слева).

zarplata = 40000
nalog = 12000
zarplata -= nalog   # Уменьшаем сумму к выдаче на размер налога
print(f"После уплаты налога: {zarplata}")  # Выдаст 28000

*= (Присвоение результата умножения)

Операция умножает значение переменной (слева) на значение выражения (справа) и присваивает результат переменной (слева).

# Вычисляем 2 в степени 12
resultat = 1
stepen = 0

while stepen < 12:
    stepen += 1
    resultat *= 2

print(f"2^12 = {resultat}") # 4096

**= (Присвоение степени числа)

Операция возводит значение переменной (слева) в степень, равную значению выражения (справа) и присваивает результат переменной (слева). Решим предыдущую задачу, используя эту операцию

resultat = 2
resultat **= 12
print(f"2^12 = {resultat}") # 4096

/= (Присвоение результата от деления)

И эта операция работает похожим образом. Она делит значение переменной (слева) на значение выражения (справа) и присваивает результат переменной (слева).

konfet = 28
bratiev = 6
sester = 1
konfet /= bratiev + sester
print(f"Каждому по {konfet} штук")  # "Каждому по 4 штук"

//= (Присвоение результата целочисленного деления)

Если конфеты не делятся нацело, остаток можно припрятать, а раздать детям по целому числу конфет.

konfet = 19
detei = 10
konfet //= detei
print(f"Каждому по {konfet} штук")  # "Каждому по 1 штук"

Задания

При решении задач надо использовать операции с присвоением.

  1. Калькулятор возраста. Программа должна запросить у пользователя текущий год и год рождения, а затем вычислять и вывести примерный возраст пользователя.
  2. Степень. Найти (путем подбора) максимальную степень числа 7, которая не превышает 7000. Например, максимальная степень числа 2, которая не превышает 20 - это число 4, т.к. 2**4 = 16 (<= 20), а 2**5 = 32 (> 20)
  3. Овердрафт. Программа имитирует оплату банковской картой, на счете которой изначально лежало 1000 руб. В цикле пользователь вводит числа (стоимость покупки), и эти числа списываются со счета. Если очередное число превышает остаток на счете, программа выдает "Овердрафт! Осталось всего...", печатая остаток счета, и прекращает работу.
