Списки

Начнем знакомство с составными типами данных в Python.

Скалярные типы данных

До этого мы рассматривали простые типы данных, ещё их называют скалярные типы, то есть типы, содержащие только одно значение. Переменная простого типа - как обычная коробка, в которой лежит всего одна вещь. Примеры:

  1. Число (например, 5)
  2. Символ (например, 'A')
  3. Логическое значение (True или False)

Список (list)

По сравнению с простыми типами данных, есть также составные типы. Они предназначены для работы с наборами данных. То есть переменная составного типа может хранить не одно значение, а целый набор. Одним из составных типов является список. Список (list) представляет тип данных, который хранит набор или последовательность элементов. В других языках программирования есть аналогичная структура данных, которая называется массив.

Создание списка

Для создания списка применяются квадратные скобки [], внутри которых через запятую перечисляются элементы списка. Например, определим список чисел:

numbers = [10, 20, 30, 40]

Подобным образом можно определять списки с данными других типов, например, определим список строк:

people = ["Tom", "Sam", "Bob"]

Список может быть пустым. Вот как его создать:

numbers1 = []

Для проверки элементов списка можно использовать стандартную функцию print, которая выводит содержимое списка в удобочитаемом виде

print(numbers)  # [1, 2, 3, 4, 5]
print(people)   # ["Tom", "Sam", "Bob"]

Если необходимо создать список, в котором повторяется одно и то же значение несколько раз, то можно использовать символ звездочки *, то есть фактически применить операцию умножения к уже существующему списку:

numbers = [5] * 6   # 6 раз повторяем 5
print(numbers)      # [5, 5, 5, 5, 5, 5]
 
people = ["Tom"] * 3    # 3 раза повторяем "Tom"
print(people)           # ["Tom", "Tom", "Tom"]
 
students = ["Bob", "Sam"] * 2   # 2 раза повторяем "Bob", "Sam"
print(students)                 # ["Bob", "Sam", "Bob", "Sam"]

Обращение к элементам списка

Значения, хранящиеся в списке, называются элементами списка.

Каждый элемент имеет свой порядковый номер в списке. Этот порядковый номер ещё называется индексом. Индексы начинаются с нуля. То есть первый элемент будет иметь индекс 0, второй элемент - индекс 1 и так далее. Доступ к элементам списка выполняется по индексу. Если надо получить второй элемент списка, то используем имя списка, после которого в квадратных скобках укажем индекс 1.

people = ["Tom", "Sam", "Bob"]
# получение элементов с начала списка
print(people[0])   # Tom
print(people[1])   # Sam
print(people[2])   # Bob

Для обращения к элементам с конца можно использовать отрицательные индексы, начиная с -1. То есть у последнего элемента будет индекс -1, у предпоследнего индекс -2 и так далее.

people = ["Tom", "Sam", "Bob"]
# получение элементов с конца списка
print(people[-2])   # Sam
print(people[-1])   # Bob
print(people[-3])   # Tom

Для изменения элемента списка достаточно присвоить элементу новое значение:

people = ["Tom", "Sam", "Bob"]
 
people[1] = "Mike"  # изменение второго элемента
print(people[1])    # Mike
print(people)       # ["Tom", "Mike", "Bob"]

Разложение списка

Python позволяет разложить список на отдельные элементы. То есть сохранить элементы списка в скалярные переменные за одну операцию присваивания:

people = ["Tom", "Bob", "Sam"]
 
tom, bob, sam = people
 
print(tom)      # Tom
print(bob)      # Bob
print(sam)      # Sam

В данном случае переменным tom, bob и sam последовательно присваиваются элементы из списка people. Однако следует учитывать, что количество переменных должно быть равно числу элементов присваиваемого списка.

Функции для работы со списком

Пока рассмотрим всего одну функцию len(list), для получения длины списка.

p = [10, 20, 30]
print(len(p))    # 3
