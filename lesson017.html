Работа со списками

Изучим на примерах самые необходимые действия, которые можно производить со списками.

Конструктор list()

Для создания списка можно применить функцию-конструктор list(). Если функцию вызвать без аргументов, она вернёт пустой список. Например, эти два определения списка аналогичны: 

numbers1 = []
numbers2 = list()

Конструктор list() может принимать набор значений (список), на основе которых создается новый список:

numbers1 = [1, 2, 3, 4, 5]
numbers2 = list(numbers1)
print(numbers2)  # [1, 2, 3, 4, 5]

Строковое значение можно превратить в список отдельных символов также используя list():

letters = list("Hello")
print(letters)      # ['H', 'e', 'l', 'l', 'o']

Сравнение списков

Два списка считаются равными, если они содержат один и тот же набор элементов (причем в том же порядке):

num1 = ["one", "two", "three"]
num2 = list(["one", "two", "three"])
if num1 == num2:
    print("Списки равны.") # Результат: "Списки равны."

Перебор элементов

Перебор элементов списка можно сделать с помощью цикла while:

people = ["Tom", "Sam", "Bob"]
i = 0
while i < len(people):
    print(people[i])    # применяем индекс для получения элемента
    i += 1

Мы пока не проходили цикл for, но надо признать, он позволяет перебирать список ещё удобнее:

people = ["Tom", "Sam", "Bob"]
for person in people:
    print(person)

В этом цикле создается переменная person, которой последовательно присваивается каждый элемент списка people. В теле цикла мы обрабатываем очередной элемент, выполняя действие над переменной person (в данном случае - выводим её значение на консоль).

Соединение списков

Для объединения списков применяется операция сложения (+), например:

abc1 = ['A', 'B', 'C']
abc2 = ['D', 'E', 'F']
result = abc1 + abc2
print(result)   # ['A', 'B', 'C', 'D', 'E', 'F']

Встроенные функции для работы со списком

Функцию len(list) мы уже проходили - она возвращает длину списка.

Функция min(list) возвращает наименьший элемент списка.

Соответственно, функция max(list) возвращает наибольший элемент:

numbers = [9, 21, 12, 1, 3, 15, 18]
print(min(numbers))     # 1
print(max(numbers))     # 21

Функция sorted(list) возвращает отсортированный список:

people = ["Tom", "Bob", "Alice", "Sam"]
 
sorted_people = sorted(people)
print(sorted_people)      # ["Alice", "Bob", "Sam", "Tom"]

Следует учитывать, что функции не меняют сам список, который передается им в качестве аргумента (в скобках). Например, функция sorted() не изменит сортируемый список, а все отсортированные элементы помещает в новый список, который возвращается в качестве результата.
